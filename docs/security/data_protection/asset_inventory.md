# Asset Inventory

An asset inventory is a core part of an ISMS and provides a complete overview of all assets that support information processing. It ensures visibility, supports risk assessments, and helps protect assets according to their value and sensitivity.

## Purpose
- Identify all organizational assets.
- Support risk assessment and classification.
- Assign ownership and responsibilities.
- Improve incident response and change management.

## Types of Assets
### Information Assets
- Customer data  
- Databases  
- Internal documents  

### Software Assets
- Operating systems  
- Applications  
- Cloud services  

### Hardware Assets
- Laptops  
- Servers  
- Network devices  

### People Assets
- Employees  
- Contractors  

### Service Assets
- External providers  
- Hosting services  

## What to Document
- Asset name  
- Asset type  
- Owner  
- Classification level  
- Location  
- Dependencies  
- Associated risks  
- Existing controls  

## Best Practices
- Keep the inventory updated regularly.  
- Assign clear ownership.  
- Automate asset discovery when possible.  
- Integrate with risk management and incident response.
