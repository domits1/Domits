### Purpose
To ensure that responsibilities related to security and compliance are clearly defined.

### Policy
| Role | Responsibility |
|------|---------------|
| *Security Manager / CISO* | Approves policies, manages audits, risk management |
| *DevOps / Cloud Engineers* | Implement AWS security controls and monitoring |
| *Management* | Provides resources and ensures compliance |
| *All Employees* | Follow security rules and report incidents |
