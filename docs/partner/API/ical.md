# iCal API Overview

This section contains Domits calendar sync documentation.

## Documents
- [iCal Import API](./ical-import.md)
- [iCal Export API](./ical-export.md)

## Scope
- Import external `.ics` feeds into Domits (`Ical-retrieve` / `ical-retrieve`)
- Export Domits event data as `.ics` (`Ical-generate`)

## Route note
- Canonical import route: `/ical-retrieve`
- Legacy compatibility route (still active): `/Ical-retrieve`
