# Domits Database ERD

## Purpose

This document tracks the updated Domits Aurora DSQL ERD.

The visual ERD is maintained in Lucidchart and is split into three readable sections:

1. Core Property ERD
2. Messaging & Integrations ERD
3. Operations, Analytics & Standalone Site ERD

## Lucidchart Reference

- Up-to-date Aurora DSQL ERD: https://lucid.app/lucidchart/f68b11d7-8ea1-42eb-80a3-2a76a1da2492/edit?invitationId=inv_1c6d44ed-fb13-413b-b392-625a493c157f&page=5ffXNMpdfGRz#

## Scope

The updated ERD covers the original property, booking, payment, amenities, rules, pricing, and availability model, plus newer Domits database areas:

- Image v2 and image variants
- Calendar prices, overrides, and iCal sources
- Cancellation policies
- Unified messaging
- Channel integrations
- Integration sync state and logs
- Property drafts, custom rules, house rules, and tasks
- KPI snapshots
- Standalone site tables
