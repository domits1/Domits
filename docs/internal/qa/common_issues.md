# Common Issues & Fixes

A quick-reference guide for recurring development problems and their solutions.


## Issue 1 — `Can't resolve ./aws-exports.js`

### Cause
`aws-exports.js` is intentionally excluded from the Git repository for security reasons.  

### Fix
1. Go to **Discord → Channel #4 → Pinned Messages**  
2. Download **`aws-exports.js`**  
3. Place it in the following directory: `frontend/web/src`
