# GitHub Onboarding Guide (Repository Access)

---

## Step 1 — Create a GitHub Account (If Needed)
If you do not already have a GitHub account:

1. Go to https://github.com
2. Click **Sign up**
3. Create an account using:
   - Email
   - Username
   - Password
4. Verify your email address

Once completed, continue to the next step.

---

## Step 2 — Open the Repository Invite Link
You will receive a **GitHub invite link**.

1. Click the **provided link**
2. Sign in to GitHub if prompted

---

## Step 3 — Accept the Repository Invitation
After opening the link:

1. Click **Accept invitation**
2. Wait a few seconds for access to be granted

You are now a member of the repository.

---

## Step 4 — Open the Repository
After accepting the invite:

1. Go to https://github.com
2. Click your **profile icon** (top right)
3. Select **Your repositories**
4. Open the assigned repository

---

## Step 5 — Verify Access
You should now be able to:
- View the repository code
- Clone the repository
- Create branches
- Open pull requests (based on provided permissions)

If you cannot perform an action, required permissions may not yet be provided.
